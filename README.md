# Stock Management System

## 📌 Description
A web-based Stock Management System built using HTML, CSS, and JavaScript.  
Allows users to manage products, perform sales (POS), and handle user registration — all in the browser using localStorage.

## 🛠 Technologies Used
- HTML/CSS
- JavaScript

## ⚙️ Features
- User login & registration
- Admin dashboard
- Product and sales management
- POS system
- Forgot password functionality

## 🚀 How to Run
Open the live system here: [Stock Management System Live](https://veshka.ct.ws)  
Or run locally:
1. Open `index.html` in your browser
2. Navigate using the homepage links
3. All data is stored in the browser
